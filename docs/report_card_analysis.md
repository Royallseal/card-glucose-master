# 《卡牌控糖师》初始卡牌数据审计报告

本报告对初始版本 20 张卡牌的数值配平、能量分布、血糖收支以及流派协同进行深度审计，旨在为项目组提供可量化的设计数据支撑。

---

## 一、 卡组结构与分布统计

### 1. 能量消耗分布
| 费用 | 卡牌数量 | 代表卡牌 |
| :---: | :---: | :--- |
| **0 费** | 4 张 (20%) | 【新鲜苹果】、【瑜伽吐纳】、【匀速长跑】、【基础降糖药】 |
| **1 费** | 13 张 (65%)| 【米饭】、【散步】、【燕麦面包】、【爽脆生菜】、【破风骑行】等 |
| **2 费** | 3 张 (15%) | 【丰盛大餐】、【极限运动】、【长效胰岛素】 |
* **平均能耗**：**0.95 费**。在每回合 3 点初始能量规则下，0.95 费的均费曲线允许单回合打出 3~4 张卡牌（如“加费+伤害+格挡”的爆发套路），为构筑提供了极高的灵活度。

### 2. 战斗属性平均能耗分布
将卡牌按实战定位（伤害卡、格挡卡、功能卡）进行归类分析，得出各战斗属性的能耗分布：
* **伤害卡 (7张)**：平均能耗 **1.14 费**。伤害卡能耗平滑（0费1张、1费4张、2费2张），【控糖胶囊】回归伤害属性，提供优秀的 1 费爆发。
* **格挡卡 (8张)**：平均能耗 **1.00 费**。生存保障稳定（0费1张、1费6张、2费1张），【速效降糖针】与【长效胰岛素】在下调升降糖幅度后，格挡分别平滑降至 23 与 35，移除了过度溢出的高格挡。
* **功能卡 (5张)**：平均能耗 **0.60 费**。由于【基础降糖药】变更为 0 费且获得 2 点能量，功能卡的均费显著下降，为爆发轮次起到了关键的加费和过牌作用。

### 3. 卡牌类型与稀有度分布
* **类型构成**：膳食卡 6张、运动卡 6张、药物卡 6张、初始卡 2张。形成对称的“三驾马车”代谢平衡架构。
* **品质分布**：初始 2张、普通 8张、良好 6张、优秀 4张。品质分布梯度平缓，保障了玩家前期开局的稳定性以及中后期构筑的追求。

## 二、 血糖收支平衡账本

### 1. 自然代谢波动（膳食与运动）
* **膳食升糖潜力**：累计 **+2.6 mmol/L**（【米饭】+0.1、【丰盛大餐】+0.6、【燕麦面包】+0.2、【新鲜苹果】+0.8、【低脂纯牛奶】+0.6、【爽脆生菜】+0.0、【无糖气泡水】+0.0）。
* **运动降糖潜力**：累计 **-3.1 mmol/L**（【散步】-0.1、【瑜伽吐纳】-0.5、【匀速长跑】-0.6、【冲刺短跑】-0.4、【花式跳绳】-0.4、【破风骑行】-0.6、【极限运动】-0.5）。
* **收支平衡审计**：膳食与运动在不使用药物时的净差为 **-0.5 mmol/L**。该数值意味着若只进行运动而不摄入膳食，血糖将逐步下滑至低血糖区间。此设定促使玩家在实战中建立“先吃碳水、后做运动”的日常代谢循环。

### 2. 药物紧急干预
* **药理降糖潜力**：累计 **-6.0 mmol/L**（【基础降糖药】-1.0、【速效降糖针】-2.0、【长效胰岛素】-3.0）。
* **药理升糖潜力**：累计 **+2.0 mmol/L**（【控糖胶囊】+2.0）。
* **干预审计**：强降糖药的总降幅经过配平优化，由原来的 -9.0 mmol/L 平抑至 **-6.0 mmol/L**，使药物卡的血糖控制更加细腻、不易导致低血糖猝死。同时，【限糖药片】能将当前血糖无条件复原至 5.7，起到强力低/高血糖保险作用。

## 四、 核心战术与数值协同

### 1. 「耐力」堆叠生存流
* **运作逻辑**：打出【低脂纯牛奶】（1费，获得 9 格挡并赋予 1 层「耐力」），使玩家接下来打出的所有格挡卡数值全部 +1（每层）。
* **战术效果**：在面对高频多段攻击或者拉长战线的战斗时，早期打出【低脂纯牛奶】积累「耐力」层数，可以极大地放大【散步】、【燕麦面包】等基础防护卡的收益，实现低能耗极高格挡的铁壁流。

### 2. 「僵硬」削弱敌方护盾
* **运作逻辑**：使用【破风骑行】（造成 12 伤害并施加 1 层「僵硬」），在对目标造成大额血量打击的同时，使其获得格挡的能力缩水 25%。
* **战术效果**：该战术适合对付高频率叠甲的强力敌人（例如重盾巨兽）。由于僵硬可以按比例严重削弱敌人的防护效率，这能间接放大玩家自身的爆发输出威力，起到绝佳的破甲效果。

### 3. 0 费大额加费与手牌爆发
* **运作逻辑**：【基础降糖药】修改为 0 费、降低 1.0 血糖、获得 2 点能量（即净获得 2 点可用费用）。
* **战术效果**：玩家可以在手牌较好、能量不足时，直接打出【基础降糖药】将能量上限在当回合内撑至 5 点。配合过牌卡【无糖气泡水】或【快速胰岛素】，可以轻松在一回合内打出诸如【丰盛大餐】（2费）+【破风骑行】（1费）+【限糖药片】（1费）的打伤害并强力重置血糖的强力 Combo。
