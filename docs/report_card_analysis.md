# 《卡牌控糖师》初始卡牌数据审计报告

本报告对初始版本 20 张卡牌的数值配平、能量分布、血糖收支以及流派协同进行深度审计，旨在为项目组提供可量化的设计数据支撑。

---

## 一、 卡组结构与分布统计

### 1. 能量消耗分布
| 费用 | 卡牌数量 | 代表卡牌 |
| :---: | :---: | :--- |
| **0 费** | 3 张 (15%) | 【新鲜苹果】、【瑜伽吐纳】、【匀速长跑】 |
| **1 费** | 14 张 (70%)| 【米饭】、【散步】、【燕麦面包】、【低脂纯牛奶】、【破风骑行】等 |
| **2 费** | 3 张 (15%) | 【丰盛大餐】、【极限运动】、【长效胰岛素】等 |
* **平均能耗**：**1.00 费**。在每回合 3 点初始能量规则下，1.00 费的均费曲线为单回合打出 3 张卡牌（“过牌+生存+调糖”连招）提供了极高的战术灵活性。

### 2. 战斗属性平均能耗分布
将卡牌按实战定位（伤害卡、格挡卡、功能卡）进行归类分析，得出各战斗属性的能耗分布：
* **伤害卡 (6张)**：平均能耗 **1.17 费**。伤害卡能耗平滑（0费1张、1费3张、2费2张），确保输出轮次不会因高费卡堆积而卡手。
* **格挡卡 (6张)**：平均能耗 **0.83 费**。中低均费的生存保障（0费1张、1费5张），允许玩家在应对重伤威胁时保留费用调糖。
* **功能卡 (8张)**：平均能耗 **1.00 费**。过牌与药理调糖件，能耗结构稳健，起到了良好的卡组润滑作用。

### 3. 卡牌类型与稀有度分布
* **类型构成**：膳食卡 6张、运动卡 6张、药物卡 6张、初始卡 2张。形成对称的“三驾马车”代谢平衡架构。
* **品质分布**：初始 2张、普通 8张、良好 6张、优秀 4张。品质分布梯度平缓，保障了玩家前期开局的稳定性以及中后期构筑的追求。

## 二、 血糖收支平衡账本

### 1. 自然代谢波动（膳食与运动）
* **膳食升糖潜力**：累计 **+2.3 mmol/L**（【米饭】+0.1、【丰盛大餐】+0.6、【燕麦面包】+0.2、【新鲜苹果】+0.8、【低脂纯牛奶】+0.6）。
* **运动降糖潜力**：累计 **-3.1 mmol/L**（【散步】-0.1、【瑜伽吐纳】-0.5、【匀速长跑】-0.6、【冲刺短跑】-0.4、【花式跳绳】-0.4、【破风骑行】-0.6、【极限运动】-0.5）。
* **收支平衡审计**：膳食与运动在不使用药物时的净差为 **-0.8 mmol/L**。该数值意味着若只进行运动而不摄入膳食，血糖将逐步下滑至低血糖区间。此设定促使玩家在实战中建立“先吃碳水、后做运动”的日常代谢循环。

### 2. 药物紧急干预
* **药理降糖潜力**：累计 **-9.0 mmol/L**（【基础降糖药】-2.0、【速效降糖针】-3.0、【长效胰岛素】-4.0）。
* **药理升糖潜力**：累计 **+2.0 mmol/L**（【控糖胶囊】+2.0）。
* **干预审计**：强降糖药的总降幅是升糖药的 4.5 倍。该设计符合人体在高血糖状态下更易受酮症酸中毒等急性病理威胁的生理事实，药物卡在卡组中充当紧急干预（刹车阀）职能。

## 四、 核心战术与数值协同

### 1. 攻防调糖组合
* **运作逻辑**：先打出【新鲜苹果】（0费，造成 8点伤害）和【低脂纯牛奶】（1费，获得 13点格挡），在仅消耗 1点能耗下完成高效输出与防御，但会带来较多升糖（共上升 1.4）。随即连打【瑜伽吐纳】（0费，获得 4点格挡，降低 0.5 血糖）和【匀速长跑】（0费，获得 1层活力，降低 0.6 血糖）。
* **战术效果**：在仅消耗 1点能量的前提下，实现 8 点输出、17 点高额防御，并将血糖升幅平抑至仅剩 0.3，实现高效攻防与血糖微调转换。

### 2. 活力多段伤害组合
* **运作逻辑**：利用【匀速长跑】叠加「活力」层数（每层使攻击伤害+1），接着打出具有多段攻击性质的卡牌。
* **战术效果**：当拥有 2层「活力」时，【新鲜苹果】（造成 4点伤害 2次）的总输出会从 8点增加到 12点；【极限运动】（造成 11点伤害 3次）的总输出会从 33点放大到 39点，极大提升了攻击效率。

### 3. 脆弱易伤爆发组合
* **运作逻辑**：先使用【冲刺短跑】（造成 9点伤害并施加 1层「脆弱」），随后打出仅需 1费的【破风骑行】（造成 14点伤害，降低 0.6 血糖）。
* **战术效果**：【破风骑行】在脆弱加成下，伤害会从 14 点直接上升至 21 点。在消耗 2点能量下，打出 30点总伤害与 -1.0 mmol/L 的降糖效果，出牌连贯性强。

### 4. 胰岛素翻倍降糖组合
* **运作逻辑**：先使用【快速胰岛素】或【长效胰岛素】为玩家施加「敏化」状态（使下一张药物的降糖效果翻倍）。
* **战术效果**：随后打出降糖药。例如配合【速效降糖针】使用，其单次降糖数值会从原有的 3.0 直接翻倍至 6.0，可以一次性清理掉大量升糖卡牌造成的负担。
