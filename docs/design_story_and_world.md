# 《卡牌控糖师》世界观与剧情设定

本文件定义了《卡牌控糖师（Card Glucose Master）》的世界观背景、主线剧情框架以及核心角色的设定依据，供剧情策划、美术设计与关卡搭建时参考使用。

---

## 一、 世界设定

### 1. 阳世界——现世之域

控糖师所生活的原初世界，是一个与当代地球高度相似的架空文明。该世界的医学与生物科技发展水平略高于现实，但尚未普遍认识到"血糖数值异常"对机体造成的深层危害。居民在长期富足的物质条件下，生活方式以高碳水化合物摄入与低运动量为特征，机体代谢负担日益累积，却未引起广泛警觉。

该世界的科技树中，存在一种名为"卡牌科技"的特殊能量传导技术——将生物信号、药理成分与运动代谢编码为实体卡牌，通过打出卡牌即可对持牌者的身体状态产生即时干预。此项技术最初仅用于民用健康管理，尚未被应用于战斗领域。

### 2. 阴世界——沦陷之境

在一次对未知空间进行的能量探勘实验中，阳世界的科学家偶然发现了一个由虫洞连接的异次元领域。该领域此前经历了比阳世界更为极端的代谢危机——其居民因长期无节制的饮食与缺乏运动，导致群体性血糖失控，最终文明崩塌。失控的代谢能量在阴世界中凝结为具象化的实体，形成了各类以代谢紊乱为特征的病变敌人。

阴世界的环境本身富含高浓度游离糖分与脂质颗粒，长期暴露会对阳世界生命体的代谢系统产生侵蚀效应。该世界的重力与大气成分与阳世界相近，但地表覆盖有粘稠的糖浆状沉积物，空气中弥漫着发酵的甜腻气息。

## 二、 主线剧情

### 1. 序幕——平静的崩裂

故事始于阳世界中一座普通的科技都市。控糖师作为一名卡牌科技研究员，正在参与一项跨次元能量通讯实验。实验过程中，虫洞意外开启，阴世界的代谢污染开始通过虫洞口渗透至阳世界——首先是空气中的糖分浓度异常上升，随后零星出现了由阴世界溢出的低级代谢实体。

当局紧急成立应对小组，控糖师因其对卡牌科技的掌握而被征召，成为抵御此次跨次元代谢危机的先锋力量。

### 2. 第一阶段——阳世界保卫战

虫洞的扩张速度超出了预期。阴世界的代谢敌人开始在阳世界的各个关键节点出现，污染当地的生态与居民健康。控糖师被派遣至受影响的区域，通过战斗逐步将敌人驱逐回虫洞另一侧。

在此阶段，控糖师逐渐掌握了利用卡牌科技在战斗中动态调控自身血糖的战斗方式，并意识到血糖的稳定程度直接决定了战斗力的上限。

### 3. 转折——虫洞的另一端

在成功封闭阳世界一侧的虫洞入口后，控糖师意识到若不从根源上解决阴世界的代谢污染问题，虫洞随时可能再次打开。经过短暂休整，控糖师毅然穿越虫洞，踏入阴世界——那个已被血糖问题彻底吞噬的沦陷之境。

### 4. 第二阶段——阴世界净化战

阴世界的环境对阳世界生命体极为不利。控糖师必须在陌生的领域中作战，面对比阳世界更为强大、更具攻击性的代谢敌人。此阶段的战斗难度急剧上升，敌人的攻击手段更为多样，且往往带有连携状态施加。

控糖师在此阶段可利用在阳世界积累的战斗经验，结合在阴世界中探索发现的新卡牌与资源，逐步深入敌人的核心领地。

### 5. 终章——双重净化

击败阴世界最深处的元凶——并发将军之后，虫洞两侧的代谢失衡将同时得到缓解。阴世界的污染浓度开始下降，阳世界的生态环境逐步恢复正常。

最终，控糖师带着从阴世界获得的知识与卡牌技术返回阳世界，将卡牌控糖技术推广至全社会，从根本上提升了整个文明对代谢健康的管理水平。

## 三、 通关条件与结局判定

### 1. 胜利条件

击败全部敌人，即可判定为通关胜利。

### 2. 失败条件

- **生命值归零**：角色生命值降至 0 时，判定为战斗死亡，战役失败。
- **血糖超限**：血糖值低于 2.0 mmol/L 或高于 15.0 mmol/L 时，判定为代谢崩溃死亡，战役失败。

### 3. 结局设定

| 结局类型 | 触发条件 | 结局描述 |
| :--- | :--- | :--- |
| **完美通关** | 击败最终 Boss | 控糖师拯救双世界，将卡牌控糖技术推广至全社会，两个世界均迎来代谢健康的新纪元。 |
| **高血糖危象** | 血糖超过 15.0 mmol/L | 控糖师在战斗中因血糖过高而导致代谢系统崩溃，战斗无法继续。 |
| **血糖缺失** | 血糖低于 2.0 mmol/L | 控糖师因血糖过低而陷入昏迷，战斗以失败告终。 |
| **力竭败退** | 生命值归零 | 控糖师在敌人的持续攻击下力竭倒下，未能完成净化使命。 |

## 四、 世界观核心冲突

本游戏的世界观冲突围绕以下三组核心矛盾展开：

### 1. 代谢平衡与紊乱

阳世界与阴世界的根本差异在于代谢平衡状态。阳世界虽有不良生活习惯普遍存在，但整体代谢系统尚未崩溃；阴世界则是代谢彻底失控后的废墟。这一矛盾贯穿整个游戏，映射于玩家的血糖管理机制之中。

### 2. 科技赋能与责任

卡牌科技作为一种中立的能量传导技术，既可用于日常健康管理，也可用于跨次元战斗。控糖师掌握这一技术后，肩负着保护两个世界的责任——技术本身不带来拯救，正确的运用方式才是关键。

### 3. 个体健康与集体命运

控糖师自身的血糖管理不仅关乎个人生死，更直接决定两个世界的命运。这一设定将"个人健康管理"这一日常行为提升至拯救世界的叙事高度，强化了游戏机制与剧情主题的统合性。

## 六、 叙事风格定位

本游戏的叙事风格定位为**轻剧情、重机制**——剧情主要服务于世界观搭建与战斗动机的赋予，不设置冗长的过场动画或分支对话。剧情信息主要通过以下途径传达：

- 开场简述：在游戏启动时以短文形式交代背景。
- 敌人图鉴：每个敌人配有简短描述，暗示其代谢异常的成因。
- 卡牌描述：卡牌名称与效果一并传达膳食、运动或药物干预的治疗逻辑。
- 结局画面：根据不同结局展示对应的叙事性画面与文字总结。
