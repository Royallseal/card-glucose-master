## 一、 v1 → v2 变更

| 项 | v1 | v2 |
| :--- | :--- | :--- |
| 意图选择 | 固定循环 | 7/12 敌人支持 `|` 双循环，开局随机选一 |
| 格挡占比 | 26% | 15% |

| 回合递增 | 无 | `TurnScaling` 每敌回合 +8%，累乘 |

## 四、 回合递增

`TurnScaling` 每敌回合 +8% 累乘。

| 敌回合 | 系数 | 将军 28 等效 | 脆×2 后 |
| :---: | :---: | :---: | :---: |
| 1 | 1.00 | 28 | 56 |
| 4 | 1.24 | 35 | 70 |
| 7 | 1.48 | 41 | 83 |
| 10 | 1.72 | 48 | 96 |
| 14 | 2.04 | 57 | 114 |

14 回时将军等效单伤 114，玩家 HP 80 + 防御约 20/回，承伤上限 4~5 次。

## 六、 结论

| 阶梯 | TTK | 承伤 | 关键 |
| :--- | :---: | :---: | :--- |
| 一层弱 | 3 | ~0 | 格挡可无损 |
| 一层强 | 4~5 | 3~8 | debuff 产生穿透 |
| 一层 Boss | 10 | 35~45 | 活力 + Scaling |
| 二层弱 | 5 | ~10 | 层间缓冲 |
| 二层强 | 8~9 | 20~28 | 高 HP + 强 debuff |
| 二层 Boss | 14 | 55~70 | 双 debuff + Scaling×2 |
