# 《卡牌控糖师》项目文档索引目录

本目录用于归纳、整理并分类导航《卡牌控糖师（Card Glucose Master）》项目的所有设计、开发、审计与历史文档。所有文档均采用相对路径互联，以便于跨平台查阅。

---

## 一、 项目规范与研发指南

本分类包含指导项目协作、代码规范、文档撰写以及功能调试的规范标准。

*   **[文档撰写风格与规范指南](./guide_documentation_style.md)**：定义项目文档的用词标准、陈述语气、Markdown 格式以及路径链接规范，指导所有文档的编修工作。
*   **[代码编写规范与设计指南](./guide_code_specification.md)**：规定 C# 代码的书写格式、命名原则、类职责划分规范以及逻辑分层要求。
*   **[开发者测试快捷键与调试指南](./developer_testing_guide.md)**：记录在 Unity 编辑器中生效的调试快捷键，包含关卡快速跳转、抽卡与爆牌测试、血糖数值强行更改及临界失败判定测试的控制说明。
*   **[项目开发与标准作业程序 (SOP)](./sop_project_dev.md)**：规范敏捷研发周期、版本分支管理、代码合并评审以及编译发布的基本流程。

## 二、 核心机制与系统设计

本分类包含关于战斗公式、卡牌结算、Buff 系统及经济玩法的核心系统蓝图设计。

*   **[状态与 Buff/Debuff 系统设计](./design_buff_system.md)**：梳理游戏内所有属性修正类（活力、耐力）、状态削减类（脆弱、乏力、僵硬）及被动次数盾类（缓释、敏化）Buff 的技术表现与结算公式。
*   **[卡牌数值公式与计算规则](./design_card_formula.md)**：详细阐述卡牌伤害、格挡的最终结算数学模型，包括血糖区间乘数与 Buff 叠加的先后计算顺序。
*   **[游戏经济与商店交易系统设计](./design_economy_system.md)**：规范关卡金币产出、商店卡牌买入与删牌移除的定价公式以及随机商品生成规则。
*   **[战斗关键字定义与设计案](./guide_combat_keywords.md)**：阐释攻击、防守、状态施加等卡牌战斗关键字的基础机制与判定时机。

## 三、 数值审计与平衡性分析报告

本分类包含对于静态 CSV 数据库编译结果的数值健康度评估，以及关卡难度梯度的科学分析报告。

*   **[代谢敌人数值与平衡审计报告](./report_enemy_analysis.md)**：提供全部 12 个敌人的属性大账、行动循环结构、TTK (预计击杀回合数) 测算，并对 `TurnScaling`（回合递增系数）成长进行收敛压力分析。
*   **[卡牌数据与稀有度分布审计报告](./report_card_analysis.md)**：统计并分析卡牌数据库的卡牌类型结构、稀有度配比合理性及核心效果分布。
*   **[数值平衡审计基本指导案](./guide_balance_audit.md)**：提供开展数值测试、定位极值卡牌以及调节关卡难度的一般性测试方法论。
